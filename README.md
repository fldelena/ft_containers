# ft_containers

## Description
The goal of this project is to understand how the STL library works, and this project is part of teaching students from 42 schools (21 schools in Moscow). The project implements the following containers of the C++98 standard:

**[List](https://www.geeksforgeeks.org/list-cpp-stl/)**
**[Vector](https://www.geeksforgeeks.org/vector-in-cpp-stl/)**
**[Map](https://www.geeksforgeeks.org/map-associative-containers-the-c-standard-template-library-stl/)**
**[Stack](https://www.geeksforgeeks.org/stack-in-cpp-stl/)**

## Additional assignments and technologies used:
* ft::pair
* ft::make_pair
* ft::lexicographical_compare
* ft::reverse_iterator
* ft::iterator
* std::allocator
* std::iterator
